Calculadora

Esta calculadora con interfaz gráfica fue desarrollada en Java Swing. Permite realizar operaciones básicas y almacenarlas temporalmente en un historial independiente para que el usuario las observe mientras continúa con otras operaciones. Además, incluye la opción de cambiar el color de fondo y una sección con información sobre el desarrollador.

Simulador Sistema Bancario

Este es un simulador de sistema bancario que ofrece funcionalidades completas como registro e inicio de sesión de usuarios, consulta de saldo, depósito y retiro de dinero. Cada transacción incluye validación de datos para garantizar la seguridad. Al finalizar cada operación, se genera automáticamente una factura detallada que incluye los datos del usuario, el monto retirado y el saldo disponible.

Sobre mí

Hola, soy Andrés!

Mi interés en la programación comenzó a los 12 años, cuando empecé a modificar videojuegos, sin saber que en realidad estaba programando. Actualmente, me dedico principalmente al desarrollo backend, con más de un año de experiencia en el campo.

Me especializo en tecnologías como Java con Spring Boot y bases de datos SQL. Además, poseo conocimientos en desarrollo frontend, utilizando tecnologías como HTML, CSS y JavaScript.

Soy estudiante de Ingeniería de Software en Bogotá, Colombia, y estoy comprometido con el aprendizaje continuo y la excelencia en el desarrollo de software.
