Calculadora en Java Swing

Esta es una calculadora con interfaz gráfica desarrollada en Java Swing. Permite realizar operaciones básicas y almacenarlas temporalmente en un apartado independiente para que el usuario las observe mientras continúa con otras operaciones. También incluye la opción de cambiar el color de fondo y una sección con información sobre el desarrollador.
